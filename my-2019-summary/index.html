蛋卷的 2019 年度总结

2019 年对于蛋卷来说，是一个重要的人生转折点。在过去的这一年中，我开启了全新的生活方式。告别了父母的管束，可以和 Grace 一起支配属于我们自己的时间。由于有了时间支配上的自由，我在人生的不同方面都有了新的发展。

一个全新的人生阶段

2019 年中和 Grace 领了证，把婚礼办了。

这是过去的一年中获得的最主要的成就。从此以后考虑问题必须要以家庭为单位，任何事情的核心不再是我自己一个人的需求。也因此感受到了被生活推着向前走的感觉。当家庭的重担压在自己身上的时候，才会学着去长大。

成熟是伴随我们一生的议题。无论何时，我都会问问自己，这样做事情的方式成熟吗？两个人住在一起以后，家庭中的琐事需要共同操心，多了许多以前接触不到的事物。我现在不会觉得多出来的那些事情很麻烦，反而觉得接触新鲜事物可以给自己很多的学习机会，它们都会成为成长的垫脚石。

第一次心理咨询

在很长一段时间里，蛋卷的焦虑一度达到了不可抑制的状态。抑郁情绪常常占据主导，与人交流时又伴随着非常强烈的争吵倾向。把所有人想的都很坏，对于自己的人生也感到失去希望。后来查了相关资料才了解到，自己的部分症状与某些心理疾病（如双相障碍）相关。

我当时感觉到自己的心理出了问题，便想寻求专业人士的帮助。而查来查去，网上推荐的许多心理咨询看上去都不那么靠谱。我又害怕自己问题真的很严重，不想去医院。最后找了一个公益心理咨询的心理医生。当时心理医生对我的安慰和认可，以及让我与自己达成和解的建议真的帮助了我很多。

很庆幸我没有真正陷入到抑郁的深渊中去。

题外话，蛋卷当时的抑郁症状不算严重。如果是很严重的症状，影响到正常的生活，或者已经开始实施自杀等行为。请及时寻求医生等专业人员的帮助。

第一次出国旅行

在初夏时节和 Grace 去了东京旅行。

第一次出国也是忐忑的，准备了详细的攻略，恶补了常用英语与日语的最基础日常用语。但在东京这样的国际大都市，对于中国游客还是非常友好的。机场和地铁站都有四种语言的标志（中英日韩），问询处也基本都有会中文的工作人员。

在东京生活的感受其实和在上海很像，当地人的生活的节奏甚至比上海更快。路上的行人大都会显出一种赶路的状态。而去到东京附近的镰仓又变成了另外一幅景象，远离繁华地带的慢节奏非常令人向往。

这一次去东京只有短短的几天，还没有办法好好地体验日本当地的文化。但仅有几天的异国风情的体验也让我们觉得，日本这个国度有着独特的魅力，很想再在日本多呆一段时间。

个人管理

从滴答清单到 Trello 再到 Notion，蛋卷的个人管理工具在短短的一年里经历了多次的迁移。中间的过程是痛苦的，而我只好以“磨刀不误砍柴工”来安慰自己。最终我得出结论，最消耗精力的是折腾工具的过程，却不是实施管理的过程。

最终，我还是把所有的内容都整理到了 Notion 中，但是我一点也不想再做迁移了。一部分原因是因为 Notion 已经足够灵活，即使产品仍然有一些缺陷，可是哪个产品不是一点点成长起来的呢。

我现在正在实行的是一套比较成熟的 GTD 系统，和正在完善中的个人笔记系统。愿新的一年，我可以更好的运用 Notion，为自己带来更多的成长。

RSS 的复兴

我自从在年初发现了 RSSHub 项目以后，我终于找到了一种自由的、开放的、极度灵活的信息获取方式。刚开始也经历了订阅大量的信息源却看不完的情况，后来也及时做了删减，有许多内容变得只看标题，也退订了一部分价值很小噪声却很大的信息源。

获取信息是每个人都有的需求。在国内各大媒体都开始转向推送各种娱乐花边的时代，如果你和我一样有着“开眼看世界的需求”，不妨也来试试 RSS 吧。

翻译

我一直对异国文化有着浓厚的兴趣。我觉得那些与国内大不相同的习俗、历史、文化有种特殊的新鲜感。因此我一直都会主动关注国外的各种动态。

有一天蛋卷在 Github 上发现了掘金翻译计划。这是一个非常有意思的项目，不仅与我的工作有关，也能满足我一部分的兴趣，和带给我成长。

在 2019 年我深度参与了掘金翻译计划，并成为了其社区管理员。上半年花了大量的时间在翻译的工作和管理项目上，有了许多收获，也渐渐遇到了瓶颈。毕竟自己的英语水平也很一般。同时，我参与的另一个非技术类翻译项目也让我感觉到重重阻碍。我明白我需要继续积累自己的英语能力，以便在将来可以更好地参与到这些项目中去。

工作

今年蛋卷获得了职称的提升。按照我领导的说法，我的能力已经足够可以称得上是高级工程师了。但其实我知道这都是表面功夫，我深知自己的技术能力还完全够不上高级，今年的工作也基本上在吃老本。

欣慰的是，今年很多的工作是对过去几年中积累的技术的补充和完善。除此之外，我也获得了架构和产品上的一些入门级提升。我不想成为全栈，但了解一些与自己工作相关的边缘知识一定对职业生涯的发展是有很大好处的。

关于 2020

对于蛋卷来说，2020 年应当是 2019 年的延续。无论是家庭、个人、还是工作，在过去的一年里有了很好的开始，那么就应当去延续。我会在新的一年里继续热爱生活，把快乐当做人生的第一要义。

新的 Flag

新的一年蛋卷的重心肯定还是会放在家庭上，毕竟家里将要迎来第三名成员了。

除了家庭以外，蛋卷还希望自己可以投入更多的精力在建立自己的个人品牌上，可以是博客、公众号，也可以是开源项目。

对于个人成长，蛋卷也需要投入更多精力在阅读和健身上。与过去不同的是，不能再以体验不同的生活为目标了，需要改为做人生的减法。

还有最后一项，那就是理财。但问题是，穷人如何理财？
